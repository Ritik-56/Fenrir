# Fenrir Frontend Design Challenge

This project was built as the part of Fenrir Security Internship Challenge.

What I implemented:-
1.Three screens in a single app
2.Interactive UI
3.Responsive design


1. Sign up Page
    i.User can manually enter the details to sign up.
    ii.Login with apple id , gmail, or meta is not currently working its just there without api integration.
2. Dashboard
    i. The dashboard is divided in two section the left side has a sidebar which has various different tabs.
    ii. Right section of the dashboard has the information about the dataset provide(currently hardcoded).
3. Active Scan Detail
    i. When clicked on any of the data it's scan report can be seen.
    ii. Spidering, Mapping, Testing, Validating, Reporting different criteria are highlighted according to the progress.

# Tech Stack
- React (Vite)
- JavaScript
- Tailwind CSS

# How to Run
npm install
npm run dev

This challenge helped me improve my understanding of React components and structures.
